## HextaUI variables

```css title="app/globals.css"  
:root {
  --hu-radius: 16px;
  --hu-background: hsl(0, 0%, 100%);
  --hu-foreground: hsl(0, 0%, 0%);

  --hu-font-inter: var(--font-inter);
  --hu-font-jetbrains: var(--font-jetbrains-mono);
}
```
