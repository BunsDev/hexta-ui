HextaUI uses a carefully selected font stack for optimal readability and performance across all devices.

- **Primary Font**: `Inter` - A modern, sans-serif typeface designed for high legibility.
- **Code Font**: `JetBrains Mono` - A monospaced font with programming ligatures for better code readability.
