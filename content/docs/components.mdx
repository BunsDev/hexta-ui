---
title: Components
description: All the components available in the Hexta UI library.
---

- [Avatar](/docs/ui/avatar) - Display user avatars.
- [Badge](/docs/ui/badge) - Show status or notifications.
- [Breadcrumb](/docs/ui/breadcrumb) - Navigation aid showing user location.
- [Button](/docs/ui/button) - Trigger actions.
- [Card](/docs/ui/card) - Display content in containers.
- [Checkbox](/docs/ui/checkbox) - Customizable checkbox with animations.
- [Chip](/docs/ui/chip) - Display small information or actions.
- [Input OTP](/docs/ui/input-otp) - Input for one-time passwords.
- [Input](/docs/ui/input) - Text input field.
- [Kbd](/docs/ui/kbd) - Display keyboard shortcuts.
- [Label](/docs/ui/label) - Label form elements.
- [Pagination](/docs/ui/pagination) - Navigate paginated content.
- [Radio](/docs/ui/radio) - Customizable radio group with animations.
- [Select](/docs/ui/select) - Select from dropdown options.
- [Separator](/docs/ui/separator) - Visually separate content.
- [Tabs](/docs/ui/tabs) - Display tabbed content.
- [Toggle](/docs/ui/toggle) - Toggle between states.
- [Tooltip](/docs/ui/tooltip) - Show additional info on hover.
