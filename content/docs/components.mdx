---
title: Components
description: All the components available in the Hexta UI library.
---

- [Avatar](/docs/ui/avatar) - A component for displaying user avatars.
- [Badge](/docs/ui/badge) - A small component for displaying status or notifications.
- [Button](/docs/ui/button) - A component for triggering actions.
- [Card](/docs/ui/card) - A component for displaying content in a container.
- [Checkbox](/docs/ui/checkbox) - A customizable checkbox component with smooth animations and multiple states.
- [Chip](/docs/ui/chip) - A small component for displaying information or actions.
- [Input](/docs/ui/input) - A component for user text input.
- [Kbd](/docs/ui/kbd) - A component for displaying keyboard shortcuts.
- [Label](/docs/ui/label) - A component for labeling form elements.
- [Pagination](/docs/ui/pagination) - A component for navigating paginated content.
- [Radio](/docs/ui/radio) - A customizable radio group component with smooth animations and flexible layouts.
- [Select](/docs/ui/select) - A component for selecting options from a dropdown.
- [Separator](/docs/ui/separator) - A component for visually separating content.
- [Tabs](/docs/ui/tabs) - A component for displaying tabbed content.
- [Toggle](/docs/ui/toggle) - A component for toggling between states.
- [Tooltip](/docs/ui/tooltip) - A component for displaying additional information on hover.
