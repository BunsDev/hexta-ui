- [Accordion](/docs/ui/components/accordion) - A component for displaying collapsible content sections.
- [Alert](/docs/ui/components/alert) - A versatile component for displaying important messages, notifications, and status updates.
- [Avatar](/docs/ui/components/avatar) - A component for displaying user profile images or icons.
- [Badge](/docs/ui/components/badge) - A small component for displaying status or notifications.
- [Breadcrumb](/docs/ui/components/breadcrumb) - A navigation component that shows the current page's location within a hierarchy.
- [Button](/docs/ui/components/button) - A clickable button component with various styles and states.
- [Calendar](/docs/ui/components/calendar) - A component for selecting dates and managing calendar events.
- [Card](/docs/ui/components/card) - A flexible container for displaying content in a card format.
- [Checkbox](/docs/ui/components/checkbox) - A component for selecting one or more options from a set.
- [Chip](/docs/ui/components/chip) - A small component for displaying tags or categories.
- [Color Picker](/docs/ui/components/color-picker) - A component for selecting colors.
- [Command Menu](/docs/ui/components/command-menu) - A component for executing commands or actions through a menu interface.
- [Date Picker](/docs/ui/components/date-picker) - A component for selecting dates with a calendar interface.
- [Drawer](/docs/ui/components/drawer) - A sliding panel component for displaying additional content or options.
- [Dropdown Menu](/docs/ui/components/dropdown-menu) - A menu component that displays a list of options when clicked.
- [File Upload](/docs/ui/components/file-upload) - A component for uploading files with drag-and-drop support.
- [Input OTP](/docs/ui/components/input-otp) - A component for entering one-time passwords.
- [Input](/docs/ui/components/input) - A basic text input component for user input.
- [Kbd](/docs/ui/components/kbd) - A component for displaying keyboard shortcuts.
- [Label](/docs/ui/components/label) - A component for labeling form elements.
- [Loader](/docs/ui/components/loader) - A component for displaying loading indicators.
- [Menubar](/docs/ui/components/menubar) - A horizontal menu component for navigation.
- [Modal](/docs/ui/components/modal) - A dialog component for displaying content in a modal window.
- [Navigation Menu](/docs/ui/components/navigation-menu) - A component for creating navigation menus with multiple levels.
- [Pagination](/docs/ui/components/pagination) - A component for navigating through paginated content.
- [Progress](/docs/ui/components/progress) - A component for displaying progress bars.
- [Radio](/docs/ui/components/radio) - A component for selecting one option from a set.
- [Resizable](/docs/ui/components/resizable) - A component for resizing elements interactively.
- [Scroll Area](/docs/ui/components/scroll-area) - A component for creating scrollable areas with custom styles.
- [Select](/docs/ui/components/select) - A dropdown component for selecting options from a list.
- [Separator](/docs/ui/components/separator) - A component for visually separating content.
- [Sidebar](/docs/ui/components/sidebar) - A component for navigation and quick access.
- [Skeleton](/docs/ui/components/skeleton) - A placeholder component for loading states.
- [Slider](/docs/ui/components/slider) - A component for selecting a value from a range.
- [Switch](/docs/ui/components/switch) - A toggle switch component for binary choices.
- [Table](/docs/ui/components/table) - A component for displaying tabular data.
- [Tabs](/docs/ui/components/tabs) - A component for organizing content into tabbed sections.
- [Textarea](/docs/ui/components/textarea) - A multi-line text input component.
- [Toast](/docs/ui/components/toast) - A flexible toast notification component built on top of Sonner with customizable styling and comprehensive functionality.
