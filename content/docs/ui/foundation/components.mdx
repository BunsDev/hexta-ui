title: Components
description: All the components available in the Hexta UI library.
icon: Component
---

- [Accordion](/docs/ui/accordion) - A component for displaying collapsible content sections.
- [Alert](/docs/ui/alert) - A versatile component for displaying important messages, notifications, and status updates.
- [Avatar](/docs/ui/avatar) - A component for displaying user profile images or icons.
- [Badge](/docs/ui/badge) - A small component for displaying status or notifications.
- [Breadcrumb](/docs/ui/breadcrumb) - A navigation component that shows the current page's location within a hierarchy.
- [Button](/docs/ui/button) - A clickable button component with various styles and states.
- [Calendar](/docs/ui/calender) - A component for selecting dates and managing calendar events.
- [Card](/docs/ui/card) - A flexible container for displaying content in a card format.
- [Checkbox](/docs/ui/checkbox) - A component for selecting one or more options from a set.
- [Chip](/docs/ui/chip) - A small component for displaying tags or categories.
- [Color Picker](/docs/ui/color-picker) - A component for selecting colors.
- [Command Menu](/docs/ui/command-menu) - A component for executing commands or actions through a menu interface.
- [Date Picker](/docs/ui/date-picker) - A component for selecting dates with a calendar interface.
- [Drawer](/docs/ui/drawer) - A sliding panel component for displaying additional content or options.
- [Dropdown Menu](/docs/ui/dropdown-menu) - A menu component that displays a list of options when clicked.
- [File Upload](/docs/ui/file-upload) - A component for uploading files with drag-and-drop support.
- [Input OTP](/docs/ui/input-otp) - A component for entering one-time passwords.
- [Input](/docs/ui/input) - A basic text input component for user input.
- [Kbd](/docs/ui/kbd) - A component for displaying keyboard shortcuts.
- [Label](/docs/ui/label) - A component for labeling form elements.
- [Loader](/docs/ui/loader) - A component for displaying loading indicators.
- [Menubar](/docs/ui/menubar) - A horizontal menu component for navigation.
- [Modal](/docs/ui/modal) - A dialog component for displaying content in a modal window.
- [Navigation Menu](/docs/ui/navigation-menu) - A component for creating navigation menus with multiple levels.
- [Pagination](/docs/ui/pagination) - A component for navigating through paginated content.
- [Progress](/docs/ui/progress) - A component for displaying progress bars.
- [Radio](/docs/ui/radio) - A component for selecting one option from a set.
- [Resizable](/docs/ui/resizable) - A component for resizing elements interactively.
- [Scroll Area](/docs/ui/scroll-area) - A component for creating scrollable areas with custom styles.
- [Select](/docs/ui/select) - A dropdown component for selecting options from a list.
- [Separator](/docs/ui/separator) - A component for visually separating content.
- [Sidebar](/docs/ui/sidebar) - A component for navigation and quick access.
- [Skeleton](/docs/ui/skeleton) - A placeholder component for loading states.
- [Slider](/docs/ui/slider) - A component for selecting a value from a range.
- [Switch](/docs/ui/switch) - A toggle switch component for binary choices.
- [Table](/docs/ui/table) - A component for displaying tabular data.
- [Tabs](/docs/ui/tabs) - A component for organizing content into tabbed sections.
- [Textarea](/docs/ui/textarea) - A multi-line text input component.
- [Toast](/docs/ui/toast) - A flexible toast notification component built on top of Sonner with customizable styling and comprehensive functionality.
